To start the server: `iex -S mix`

Examples testcases:
  ```elixir
  Cache.start_link()
  Cache.register_function(fn-> Process.sleep(1000); {:ok, System.os_time(:second)} end, "key", 10000, 1000)
  Cache.get("key")
  ```
